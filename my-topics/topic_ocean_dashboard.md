# 個人題目探索：海洋環境資料儀表板

## 題目名稱

海洋環境資料儀表板

## 為什麼對這個題目有興趣

海洋環境資料包含海溫、潮位、浪高、風速與鹽度等欄位，這些資料能反映不同海域與不同日期之間的變化。相較於單純閱讀 CSV 表格，若能把資料整理成互動式儀表板，就可以更直覺地觀察測站差異與時間趨勢。

這個題目也很適合結合巨量資料與雲端運算課程中學到的內容，包含 Python 資料分析、資料視覺化、Gradio Web 介面與 Docker 部署。透過這個專題，可以練習從資料處理到系統展示的完整流程。

## 可能使用的資料來源

| 資料來源 | 說明 |
| --- | --- |
| 本專題 sample CSV | 期末專題展示用模擬資料 |
| 中央氣象署海象資料 | 未來可作為真實資料來源參考 |
| 海洋觀測公開資料 | 未來可串接 API 或下載真實觀測資料 |

目前專題先使用 `data/raw/sample_ocean_data.csv` 作為展示資料，欄位格式參考海洋觀測資料常見設計。

## 預計使用的技術

- Python：主要程式語言。
- Pandas：讀取與清洗 CSV 資料。
- NumPy：進行基本統計分析。
- Matplotlib：產生折線圖與長條圖。
- Gradio：建立互動式資料儀表板。
- Docker：將系統容器化，方便部署與 Demo。

## 預期完成成果

完成後，使用者可以透過瀏覽器開啟海洋環境資料儀表板，選擇範例資料或上傳 CSV，並查看資料預覽、清洗結果、統計摘要、測站分組統計與視覺化圖表。系統也能透過 Docker 啟動，方便在不同電腦上重現執行環境。
